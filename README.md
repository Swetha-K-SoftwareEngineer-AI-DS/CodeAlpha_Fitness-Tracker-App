# Fitness Tracker App

## Overview

The Fitness Tracker App is a simple web application that helps users track their daily fitness activities. Users can manually log their steps, exercise type, workout duration, and calories burned. The app provides an easy-to-use interface for monitoring fitness progress.

## Features

* Add daily fitness activities.
* Track steps, exercise type, workout duration, and calories burned.
* Display activity records on the screen.
* Input validation to ensure all fields are filled.
* Simple and responsive user interface.
* Built using HTML, CSS, and JavaScript.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Install the **Live Server** extension.
4. Right-click on `index.html`.
5. Select **Open with Live Server**.
6. The application will open in your browser.

## Usage

1. Enter the number of steps.
2. Enter the exercise type.
3. Enter workout duration in minutes.
4. Enter calories burned.
5. Click the **Add Activity** button.
6. The activity details will be displayed under the activity list.
